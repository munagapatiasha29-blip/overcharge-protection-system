# Overcharge Protection System ⚡

## 🔗 Live Demo

## 📌 Project Description
This project simulates a battery overcharge protection system. It helps prevent damage by stopping charging when the battery reaches a safe limit.

## 🚀 Features
- Detects overcharging condition
- Stops charging automatically
- Simulation-based output
